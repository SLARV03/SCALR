# SCALR
PBL Project (Compiler Design)

### Testing Guide
1. run the makefile
2. For windows: .\test_parser.bat example_grammar.txt SLR1
3. For linux: .\test_parser.sh example_grammar.txt SLR1

SLR1 can be replaced by your parser to test, the output comes out in a json file (all the tables, good luck reading that).The grammar is currently stored in example_grammar.txt which is being used for string input
